SBS Go (Weiqi) Widget Prototype

This prototype demonstrates the Go widget with different themes, sizes, and interactive modes.

Book Theme (19x19)

Pure black and white style, suitable for digital books and printing.

Classic Theme (19x19)

Traditional wood board style with shadows.

9x9 Static (Setup & Markers)

Compact 9x9 board with various markers (Triangle, Square, Circle, Labels) on stones and empty spots.

13x13 Interactive (All Numbers)

Medium size 13x13 board showing all move numbers.

19x19 Interactive (Last 10 Numbers)

Large 19x19 board showing move numbers only for the last 10 moves.

19x19 Static (No Coordinates)

Static display without coordinates, clean book style.

19x19 Extra Large (1000px)

Testing the rendering quality and layout at a much larger scale.
