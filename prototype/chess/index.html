SBS Chess Widget Prototype

This playground exercises the future `sbs-chess` extension before it is promoted into the shared widgets bundle. Each example mirrors the workflow documented in README.md.

Phase 1 focuses on accurate board rendering + move playback scaffolding. Interactive move entry and legality checks will layer on once visuals stabilize.

Language:
480px

Demo 1 · Starting Position (Static)

Default configuration with the initial FEN, static orientation, and controls hidden.

Demo 2 · Immortal Game Highlights

Move list provided as an array with intermediate FEN snapshots to step through a famous attacking sequence.

Demo 3 · Black Orientation + Compact Size

Board rendered from a composed FEN facing Black with a reduced footprint, showing how authors can tune layout.

Demo 4 · Move Log Only (No Snapshots)

Demonstrates graceful degradation when only SAN/figurine text is supplied—the board stays on the baseline FEN while the move list remains navigable.

Demo 5 · Minimal Board

Only the board, pieces, and coordinate guides are visible—useful for tight layouts or inline diagrams.

Demo 6 · Half-Size Board

Demonstrates the `size` parameter by rendering a full widget at 50% of the default board dimensions.

Next steps: add interactive move entry, legal move validation, undo/redo, and annotation overlays.
